# Instructions
1) Edit config file withe the urls and paths of your models from Huggingface
2) Create empty models folder: `mkdir ./models`
3) `pip install -r requirements.txt`
4) Launch chatbot:
    - Terminal-based interface:
    `python main.py`
    
    - Tkinter GUI:
    `python gui.py`

Tested on Linux, no GPU.
